# VibeBuild

VibeBuild 是一个由 AI 驱动的创新协作与黑客松生态平台。

## 项目结构

- `frontend/`: React + TypeScript + Vite 前端项目
- `backend/`: FastAPI 后端项目
- `docs/`: 项目文档 (包含 PRD)

## 快速开始

### 前端

```bash
cd frontend
npm install
npm run dev
```

### 后端

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
