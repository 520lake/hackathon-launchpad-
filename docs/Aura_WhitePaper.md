> **版本**: 1.0.0  
> **日期**: 2026-02-01  
> **定位**: 中国版 Devpost —— AI 赋能的黑客松创新协作平台

---

## 1. 产品简介 (Product Introduction)

**Aura** 是一个专为开发者、组织者和评委打造的**一站式黑客松管理与协作平台**。我们致力于解决传统黑客松活动中“组织难、组队难、评分难”的痛点，通过集成先进的 AI 大模型能力（ModelScope/DashScope），为黑客松的全生命周期提供智能化赋能。

不同于市面上常见的 SaaS 工具，Aura 采用了极具辨识度的 **Brutalist（粗野主义）** 设计风格，强调“去 AI 味”的硬核科技感，回归代码与创新的本质。

## 2. 核心功能 (Core Features)

### 2.1 🤖 AI 深度赋能 (AI-Empowered)
Aura 利用大模型能力渗透到黑客松的各个环节，实现真正的智能化辅助：

- **🔮 AI 社区洞察 (Community AI Insights)**:
  - 在“参赛者广场”实时生成社区分析报告。
  - 自动分析所有报名者的技能分布、兴趣聚类与人口统计特征。
  - 为未组队用户提供宏观的组队建议与趋势指引。
- **⚡ AI 智能组队 (AI Team Match)**:
  - 位于活动详情的“社区与组队”版块。
  - 基于用户的技能标签、性格特质（Personality）与项目偏好，智能推荐最契合的队友。
  - 提供详细的匹配理由（如“技能互补”、“性格相投”），打破陌生人社交壁垒。
- **✨ AI 项目副驾驶 (AI Project Copilot)**:
  - 在项目提交阶段提供 AI 增强功能。
  - 根据简短的想法草稿，自动生成专业的项目简介、商业计划书概要与核心亮点描述。
  - 辅助生成项目展示文案，提升项目在评审中的竞争力。
- **🏗️ AI 活动架构师 (AI Event Architect)**:
  - 组织者只需输入一句话主题（如“生成式 AI 辅助教育”）。
  - AI 自动生成完整的活动策划案、赛题维度、评分标准与宣传物料。

### 2.2 🎯 面向人群与全生命周期管理 (Target Audience & Lifecycle)
Aura 精准服务于黑客松生态中的三类核心人群：

#### 1. 组织者 (Organizer) —— “从繁杂事务中解脱”
- **痛点**: 策划耗时、运营成本高、评审统计难。
- **功能**:
  - AI 一键策划与规则生成
  - 可视化报名管理与数据看板
  - 自动化评审流程配置

#### 2. 参赛者 (Participant) —— “专注于创新本身”
- **痛点**: 找不到队友、项目文案弱、缺乏灵感。
- **功能**:
  - **社区与组队**: 利用 AI 寻找神队友，查看社区大盘数据。
  - **我的项目**: 赛后持续打磨项目，利用 AI 优化商业叙事。
  - **参赛者广场**: 透明化的社区展示，快速建立联系。

#### 3. 评委 (Judge) —— “高效精准的决策”
- **痛点**: 项目数量多、评分维度复杂、主观偏差大。
- **功能**:
  - 沉浸式评审终端 (Judging Terminal)
  - AI 辅助生成评语与评分建议
  - 多维度雷达图打分体系

### 2.3 🎨 极致 UI/UX 设计
- **Brutalist 风格**: 高对比度、大字体、硬朗边框、不对称布局，打造独特的极客视觉体验。
- **多语言支持**: 完美支持中文/英文双语切换，服务全球开发者。
- **响应式交互**: 优化的移动端适配，随时随地管理活动。

## 3. 技术方案特色 (Technical Architecture)

### 3.1 总体架构
采用现代化的前后端分离架构，容器化部署，确保高可用与易扩展。

| 模块 | 技术选型 | 优势 |
| :--- | :--- | :--- |
| **前端** | React + TypeScript + Vite | 高性能、类型安全、开发体验极佳 |
| **样式** | Tailwind CSS v4 + GSAP | 原子化 CSS 引擎，配合 GSAP 实现流畅动画 |
| **后端** | FastAPI (Python) | 异步高性能，原生支持 OpenAPI 文档，易于 AI 集成 |
| **数据库** | SQLModel (SQLAlchemy) + SQLite/PG | ORM 现代化封装，支持多种数据库后端切换 |
| **AI 引擎** | ModelScope (Qwen-Plus) | 接入阿里通义千问大模型，国产化支持友好 |
| **部署** | Docker + Docker Compose | 一键部署，环境一致性保障 |

### 3.2 亮点技术细节
1.  **单容器全栈部署**: 
    - 针对魔搭（ModelScope）等 PaaS 平台优化，通过 FastAPI 托管静态资源，实现前后端在同一端口（7860）运行，降低部署成本。
2.  **数据持久化方案**:
    - 智能识别运行环境，在云端自动挂载持久化存储（`/mnt/workspace`），确保 SQLite 数据库文件不因容器重启而丢失。
3.  **双重认证体系**:
    - 支持 **微信扫码（测试号）** 与 **邮箱验证码** 双重登录方式，适应国内用户习惯。

## 4. 商业价值与愿景 (Vision)

Aura 不仅仅是一个工具，更是一个**创新生态连接器**。
- **对开发者**: 提供展示才华的舞台，降低参与门槛。
- **对企业/组织**: 降低举办黑客松的成本，高效筛选优秀人才与项目。
- **对社区**: 沉淀优质开源项目，活跃技术氛围。

我们的目标是成为**中国最大的黑客松基础设施平台**，让每一次灵感的迸发都能找到落地的土壤。
